How ddbya Desktop works

Ddbya intercepts your calls to your Anthropic-compatible API provider and tracks token usage as your session progresses. It acts as a local proxy, forwarding requests with the same authentication you would use normally. Anthropic Subscriptions are not supported — only direct API access.

First-time setup

Enter your provider's API base URL in the field above (e.g. https://api.anthropic.com). Then, in Claude Desktop, open the Developer menu (normally hidden) and choose Config Third-Party Inference → Connection. Select Gateway from the drop-down at the top. Under Gateway Credentials, set Gateway base URL to the address below, then configure the rest as you would for your real provider.
